# demo-compute-modules-scraper-milwaukee-ts


1. **Run startup.sh**
    ```bash
    ./startup.sh
    source startup.sh
    ```
2. **Install Dependencies**
   ```bash
   npm install
   ```


## Usage

1. **Run the Scraper**
   ```bash
   npm start
   ```

2. **Run with Debug Output**
   ```bash
   DEBUG=1 npm start
   ```

3. **Run Tests**
   ```bash
   npm test
   ```

4. **Build JavaScript Files**
   ```bash
   npm run build
   ```
